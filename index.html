Gone Phishing

Work in Progress: Gone Phishing is a web browser based game that aims to educate and enhance users' confidence in thwarting phishing attacks.

This project was developed by a team of 6 using C#, JavaScript, HTML, Firebase, Unity and WebGL.

MoreTea

"Search Engine Enhancer"

"Search Engine Enhancer" is a Chrome extension that was developed for Terrible Ideas Hackathon 2021.

This project was developed by a team of 4 using JavaScript, HTML and CSS.

Check out the Github repo to download and try it out yourself.

Disclaimer: The extension doesn't actually enhance your search engine. You have been warned.

Speedy Car

Speedy Car is a web browser based game that was developed during Kiwijam 2021 for the theme "Chain Reaction".

This project was developed as a team of 4 using C# and Unity.

Click here to try the game!

QR Code Detector

The QR Code detector takes an image that contains a non-rotated QR code and draws a box around the QR code.

It was developed in Python using the library matplotlib as an assignment for CompSci 373: Computer Graphics and Image Processing at the University of Auckland.
